### C++ Project Assignment – Individual Work <br> 

Each student is required to complete an individual programming project using **C++**, based on the topics covered in the course.
---
### Objective
Apply the concepts learned in class (such as variables, control structures, functions, arrays, structures, etc.) to build a practical and well-organized C++ program.
---
### Requirements
1. **Choose a Project Topic**
   * The topic must be related to the course material.
   * Examples:
     * Student Management System (using `struct`)
     * Simple Banking System
     * Library Management
     * Quiz Game
     * Any creative idea using functions + arrays/structures

2. **Create Your Own Repository**
   * Use a version control platform such as GitHub or GitLab.
   * The repository must include:
     * Source code (`.cpp` files)
     * Clear file structure
     * Meaningful commit history

3. **Documentation**
   * Add a `README.md` file that includes:
     * Project description
     * Features
     * How to run the program
     * Sample output (optional)

4. **Submission Process**
   * Submit your repository link using the provided Google Form.
   * **IMPORTANT:** Keep your repository **private until the deadline**.
   * After the deadline, make your repository **public**.

5. **Presentation**
   * Each student will present their project during the lab session.
   * Be ready to:

     * Explain your code
     * Demonstrate how it works
     * Answer questions
